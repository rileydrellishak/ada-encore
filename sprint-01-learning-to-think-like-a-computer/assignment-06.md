# 🔁 Working with Collections of Data

## 🌱 Why are we doing this?

Real programs rarely deal with single values.

They deal with:

* lists of users
* collections of data
* groups of related information

This assignment is about learning how to think in *batches of data*, not just individual values.

---

## 📚 Learning Objectives

By the end of this assignment, you will be able to:

* Iterate over lists using loops
* Access and manipulate dictionary data
* Apply logic across multiple items
* Use for-loops to process collections
* Recognize patterns in grouped data

## 📝 Assignment Spec

---

### Part 1 — List processing

Write a program that:

* takes a list of values
* processes each item
* produces a new list OR aggregated result

---

### Part 2 — Dictionary processing

Work with at least one dictionary structure:

Example:

```python id="dict_example"
student = {
    "name": "Ada",
    "scores": [90, 85, 88]
}
```

Perform operations like:

* extracting values
* transforming values
* computing summary results

---

### Part 3 — Pattern recognition

Write a short explanation:

* What patterns did you notice in the data?
* How did iteration help reveal structure?

---

## ⚠️ Common Pitfalls

* modifying a list while iterating over it
* treating dictionaries like lists
* writing overly complex loop logic
* forgetting to initialize accumulators

---

## ✅ Acceptance Criteria

* You used at least one for-loop over a list
* You processed at least one dictionary
* You produced a transformed result from input data
* Your code is readable and structured
* You can explain what each loop is doing
